## ğŸ”§ TROUBLESHOOTING

### "npm: command not found"
**Solution**: Install Node.js from nodejs.org

### "Can't login"
**Solution**: Check spelling!
- Username: `boy` or `girl` (lowercase!)
- Password: `love123` (with numbers!)

### "Port 3000 already in use"
**Solution**: Use different port
```powershell
npm run dev -- -p 3001
```

### "Page is blank"
**Solution**: 
- Refresh browser (F5)
- Check PowerShell says "ready"

### "Messages not appearing"
**Solution**:
- Use different usernames in each window
- Refresh browser (F5)
- Clear browser cache
